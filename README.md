# 🎨 Patternly
**PATTERNLY** est un éditeur de patrons d'impression complet ! 

---
Fonctionnalités incluses :

## **🎨 Formes (20 types)**
Rectangles, cercles, ellipses, triangles, étoiles (4/5/6 branches), hexagone, pentagone, octogone, losange, cœur, flèche, croix, ligne, texte, lune, vague, zigzag

## **✏️ Édition complète**
- Déplacer, redimensionner, pivoter, retourner (H/V)
- Couleur de fond, contour, opacité, épaisseur de trait
- Dégradés (horizontal, vertical, diagonal, radial)
- Ombres portées (couleur, flou, décalage)
- Styles de trait (plein, tirets, pointillés)
- Arrondi des coins pour les rectangles
- Propriétés texte (police, taille, gras/italique/souligné)

## **📐 Outils avancés**
- Alignement (gauche/droite/haut/bas/centre H+V)
- Distribution régulière H/V
- Ordre des calques (avant/arrière)
- Grouper/dégrouper
- Magnétisme sur grille + grille visuelle (lignes ou points)

## **🔄 Modes de répétition (8)**
Répétition simple, brique H/V, demi-chute, miroir H/V/H+V, rotation 90°/180°

## **💾 Workflow**
- Annuler/Rétablir illimité
- Copier/Coller
- Calques avec visibilité et verrouillage
- Sauvegarde projet JSON + rechargement
- **Export PNG/JPEG/WebP** en résolution 1×/2×/4× ou personnalisée
- Aperçu plein écran du motif en temps réel

### **⌨️ Raccourcis clavier**
`R` rect · `C` cercle · `T` texte · `V` sélection · `H` pan · `G` grille · `S` magnétisme · `P` aperçu · `Ctrl+Z/Y` annuler · flèches pour déplacer
